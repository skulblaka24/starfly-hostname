Hostname
========

Ce role configure le hostname et modifie /etc/hosts et /etc/hostname.

Requirements
------------

None.

Platform
--------

Debian ou RHEL

Role Variables
--------------

<pre><code>hostname: HOSTNAME.starfly.ovh</code></pre>

Dependencies
------------

Ce role se télécharge automatiquement à partir du requirements.yml
